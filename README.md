# Barrio Burrito POS

Offline Android tablet POS application for **Barrio Burrito** built with:

* Kotlin
* Jetpack Compose
* Room Database

## Features

* Tablet-friendly POS UI
* Menu and current order panel
* Offline local data storage
* Order history
* Inventory management
* Daily sales reports
* Settings page

## Project Goal

This project is designed for a small mobile food cart business that sells items like fries, juice, burrito, nachos, and other snacks using an Android tablet in an offline environment.

## Tech Stack

* **Language:** Kotlin
* **UI:** Jetpack Compose
* **Database:** Room
* **Architecture:** MVVM

## Status

Currently in development.
